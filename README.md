# react-flask-mongo-app

Sample dev app for testing stuff

## Super simple example

from the example here: [https://dev.to/nagatodev/how-to-connect-flask-to-reactjs-1k8i](https://dev.to/nagatodev/how-to-connect-flask-to-reactjs-1k8i)

    ```bash
    cd flask_react

    # setup virtual environment with flask & python-dotenv
    python3 -m venv .venv

    # start the flask backend
    npm run start-backend
    # start react frontend
    npm start
    ```
